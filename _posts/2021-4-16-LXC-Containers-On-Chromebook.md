PowerBI makes it super easy to ingest automated report output (from practically any cloud storage source) and visualize thousands of rows of mind numbing data easily. Here are my notes that solves hours of searching for an answer.

1. Automation outputs a CSV filename like 'important-report - 03302021.csv' (mmddyyyy) but the CSV data does not include a date column. After getting the data, duplicate the 'Source.Name' column, Replace Values to clean the filename down to just the sdate '03302021'. Use the Add Column tab to add a Custom Column, then use this formula as a starting point to get text that is formatted properly so that changing the data type to Date completes successfully. `Text.Combine({Text.Start([#"Source.Name - Copy"], 2), "/", Text.Middle([#"Source.Name - Copy"], 2, 2), "/", Text.Middle([#"Source.Name - Copy"], 4)}), type text)` The source for this example came from [https://stackoverflow.com/questions/57392081/power-bi-convert-text-yyyymmdd-to-date-dd-mm-yyyy](https://stackoverflow.com/questions/57392081/power-bi-convert-text-yyyymmdd-to-date-dd-mm-yyyy)
