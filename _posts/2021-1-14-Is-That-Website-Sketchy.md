It is easier than ever for anyone to create a website. However, it is easier than ever for someone to create a website that intends to do someone harm. To begin figuring out is a particular website is sketchy, here are some links.

Level
1. What does the Information Security industry say?

    [https://global.sitesafety.trendmicro.com/](https://global.sitesafety.trendmicro.com/)

    [https://sitecheck.sucuri.net/](https://sitecheck.sucuri.net/)

    [https://safeweb.norton.com/](https://safeweb.norton.com/)

    [https://www.virustotal.com/gui/](https://www.virustotal.com/gui/)

    [https://webscan.upguard.com/](https://webscan.upguard.com/)

2. Is the host or domain on a blocklist or have a technical issue triggering a security false positive?

    [https://mxtoolbox.com/domain](https://mxtoolbox.com/domain)

    [https://www.spamhaus.org/lookup/](https://www.spamhaus.org/lookup/)

    [https://www.ssllabs.com/ssltest/](https://www.ssllabs.com/ssltest/)

That is it. If you are still testing at this point, that website is sketchy.
